Requirements

To run APlayer you need a machine, either a PowerPC or x86, capable of running BeOS Release 5.x. It should't be too demanding on the cpu(s).

APlayer is developed and tested with:

Dual Pentium III - 933MHz + 512Mb RAM
Pentium II 800MHz + 512Mb RAM

Of course it does not need that fast computer...

Apple 4400 (PPC603 200MHz) is used to compile PPC version.

Other tested computers:

Celeron 458MHz + 256Mb RAM
